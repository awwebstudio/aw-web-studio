Portofolio AW Web Studio

Kumpulan project website yang saya desain dan kembangkan untuk kebutuhan toko online, UMKM, fashion, jasa jahit, dan bisnis lokal.

Teknologi yang Saya Gunakan

Seluruh website dalam portofolio ini dikembangkan menggunakan Laravel, PHP, MySQL, HTML, CSS, dan JavaScript dengan tampilan responsif yang dapat digunakan pada desktop maupun perangkat mobile.

Laravel PHP MySQL Laragon HTML CSS JavaScript Responsive Design
